# Fireball

The Fireball is a ranged weapon that launches a fiery projectile towards enemies, dealing significant damage upon impact. It is effective against groups of enemies due to its area-of-effect damage.

## Aspect Table

| Aspect    | Description    |
| --------- | -------------- |
| Targeting | PROJECTILE/AOE |
| Range     | Ranged         |
| Cooldown  | 4sec           |
| Scaling   | Damage         |
| Status    | None           |

## Playstyle

Due to its slow attack but hard hit the player has to make sure he can group up enemies in between hits. This should result in a risk/reward playstyle.
