# Enemy Types

Currently there are five main types of enemies in the game:

1. **Walker**: Basic ground enemy that moves slowly towards the player.
2. **Sprinter**: Fast-moving enemy that can quickly close the distance to the player
3. **Jumper**: Enemy that can leap over obstacles to reach the player.
4. **Shooter**: Ranged enemy that attacks the player from a distance.
5. **Swarm**: Small, numerous enemies that attack in groups.
